# Student Registration Form

A beginner-friendly full-stack web application for student registration, built with Node.js, Express, and EJS. This project demonstrates how to handle basic web forms, process POST requests, and render dynamic HTML pages using a templating engine.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, EJS (Embedded JavaScript Templating)
- **Backend:** Node.js, Express.js
- **Design:** Modern CSS (Flexbox, Card Layouts, Custom Fonts)

## ✨ Features

- **Dynamic Form Handling**: Successfully captures and processes user input via `POST` requests.
- **Form Validation**: Implements basic HTML5 validation for required fields, email formatting, and age restrictions.
- **Server-Side Rendering**: Uses EJS to dynamically generate the success page with the submitted user data.
- **Responsive UI**: Features a clean, centered card layout that looks great on both desktop and mobile devices.
- **Interactive Design**: Includes smooth button hover effects and custom modern typography (Poppins font).

## 🚀 Installation Steps

To set up this project locally on your machine, follow these steps:

1. **Clone the repository** (if applicable) or download the project files.
2. **Open your terminal** and navigate to the project folder.
3. **Initialize the project** (if `package.json` is not present):
   ```bash
   npm init -y
   ```
4. **Install the required dependencies**:
   ```bash
   npm install express ejs
   ```

## 💻 How to Run the Project

1. Ensure you have installed the dependencies using the steps above.
2. Start the Express server by running the following command in your terminal:
   ```bash
   node server.js
   ```
   *(Alternatively, you can run `npm start`)*
3. Open your web browser and navigate to:
   ```text
   http://localhost:3000
   ```
4. Fill out the registration form and click submit to view your dynamic result page!
